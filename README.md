# Nifty100 Financial Analytics Platform

## Project Overview

This project builds a complete financial analytics platform using Nifty100 company data. It includes data extraction, validation, transformation, storage in SQLite, and preparation for analytics and dashboards.

## Tech Stack

- Python
- Pandas
- SQLite
- SQL
- Git & GitHub
